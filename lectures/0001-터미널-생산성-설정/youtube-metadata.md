## 📝 설명 (Description)

> 아래 전체 텍스트를 YouTube 설명란에 붙여넣기 합니다.

```
개발자라면 하루에 수십~수백 번 터미널을 사용합니다. 터미널 환경을 제대로 세팅하면 연간 44시간을 절약할 수 있습니다.
이 강의에서는 macOS 터미널을 처음부터 끝까지 최적화하는 방법을 실시간 데모로 보여드립니다.

⏱️ 타임스탬프
00:00 인트로 — 왜 터미널을 최적화해야 하는가
00:05 개요 — 시간 절약 데이터로 설득하기
00:10 시작하기 — Homebrew, iTerm2, Oh My Zsh 설치
00:20 핵심 개념 — 터미널/셸/CLI, Zsh vs Bash
00:25 필수 도구 설치 — Powerlevel10k, 개발 폰트, iTerm2 고급 설정
00:35 생산성 플러그인 — autosuggestions, syntax-highlighting, z, fzf
00:45 현대적 CLI 도구 — exa, bat, fd, ripgrep, git-delta, htop, tldr

✅ 이 강의에서 배우는 것
• iTerm2 + Oh My Zsh + Powerlevel10k로 시각적으로 강력한 터미널 구성
• zsh-autosuggestions + fzf로 명령어 입력 시간 82% 단축
• exa, bat, fd, ripgrep 등 현대적 CLI 도구로 파일/텍스트 검색 5배 향상
• Git 워크플로우 자동화 — gcp, gcof 함수로 commit+push 한 줄 완성
• dotfiles + Brewfile로 어떤 Mac에서도 1시간 안에 환경 복원
```
