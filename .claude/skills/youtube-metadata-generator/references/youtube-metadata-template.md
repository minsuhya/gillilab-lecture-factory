## YouTube SEO 가이드라인

### 제목 작성 원칙

| 원칙 | 설명 |
|---|---|
| 60자 이내 | 검색 결과에서 잘리지 않는 최대 길이 |
| 핵심 키워드 앞쪽 배치 | 앞 30자가 검색 알고리즘에 더 중요 |
| 숫자 활용 | "5가지", "10분", "82% 단축" 등 구체적 수치 |
| 과장 지양 | 실제 내용을 정확히 반영 (클릭베이트 페널티 위험) |
| 파이프(|) 구분 | 키워드 | 부연설명 형식으로 가독성 향상 |

### 설명 작성 원칙

| 구간 | 권장 분량 | 목적 |
|---|---|---|
| 훅 (첫 2~3줄) | 150자 이내 | 펼치기 전 노출되는 영역, 클릭 유도 |
| 타임스탬프 | 섹션 수만큼 | 시청자 편의, 챕터 기능 활성화 |
| 학습 내용 | 3~5개 | 시청 결정 요소 |
| 관련 링크 | 강의 언급 도구 | 신뢰도 및 편의성 |
| 해시태그 | 3~5개 (맨 끝) | 해시태그 페이지 노출 |

> **타임스탬프 챕터 활성화 조건:** 타임스탬프가 3개 이상이고 첫 번째가 `00:00`이어야 합니다.

### 태그 전략

| 유형 | 예시 | 비중 |
|---|---|---|
| 브로드 (넓은 범위) | 개발, 프로그래밍, 코딩 | 20% |
| 미드 (중간 범위) | 터미널, macOS, 맥개발 | 40% |
| 롱테일 (구체적) | iTerm2설정, OhMyZsh플러그인 | 40% |

### 썸네일 디자인 원칙

| 원칙 | 설명 |
|---|---|
| 해상도 | 1280×720px (16:9, 최소 640px 너비) |
| 파일 크기 | 2MB 이하 |
| 폰트 크기 | 모바일에서도 읽힐 수 있는 크기 (최소 40pt 권장) |
| 색상 대비 | 배경과 텍스트 간 명확한 대비 |
| 얼굴 포함 | CTR 향상에 효과적 (선택사항) |
| 브랜드 일관성 | 채널 전반에 걸쳐 동일한 스타일 유지 |

### 최종 화면 & 카드 가이드

| 요소 | 권장 시점 | 목적 |
|---|---|---|
| 최종 화면 | 영상 마지막 20초 | 시청 지속, 구독 유도 |
