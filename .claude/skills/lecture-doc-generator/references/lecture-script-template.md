# lecture-script.md 전용 템플릿

강의자가 화면에서 명령어를 실행하며 녹화하는 라이브 코딩 형식의 낭독 스크립트 템플릿입니다.

---

## 형식 규칙 요약

| 항목 | 규칙 |
|---|---|
| 문체 | ~합니다 격식체 엄수 |
| 청중 참여 문구 | 금지 (서술형으로 전환) |
| 코드 실행 전 | "화면에서 다음 명령어를 실행합니다." |
| 코드 실행 후 | "실행하면 [결과]를 확인할 수 있습니다." |
| 총 분량 | 60~90분 낭독 분량 |

## 큐 마커 사용법

| 마커 | 사용 시점 |
|---|---|
| `[인트로]` | 강의 오프닝 |
| `[📌 섹션 전환]` | 새 섹션 시작 시 |
| `[💻 데모]` | 터미널/에디터 실행 구간 |
| `[⏸️ 잠깐]` | 주의사항/부연 설명 |
| `[🎯 핵심 포인트]` | 강조해야 할 핵심 내용 |
| `[📝 코드 설명]` | 코드 라인별 상세 설명 |

---

## 템플릿

```markdown

오늘 강의에서 다룰 내용입니다:

- {항목 1}

## 문체 변환 빠른 참조

| 비격식체 (금지) | 격식체 (사용) |
|---|---|
| `~볼게요` | `~보겠습니다` |
| `~해볼게요` | `~해보겠습니다` |
| `~이에요` / `~예요` | `~입니다` |
| `~거든요` | `~기 때문입니다` |
| `~잖아요` | `~습니다` (문맥에 맞게) |
| `~돼요` | `~됩니다` |
| `~있어요` / `~없어요` | `~있습니다` / `~없습니다` |
| `~되는 거죠` | `~됩니다` |
| `~이거` | `~이것` |
| `~주세요` | `~하시기 바랍니다` |
| `따라해보세요` | (제거) |
| `~계신 분 있으시죠?` | (제거 또는 서술형 전환) |

## 청중 참여 문구 → 서술형 변환 예시

| 금지 (청중 참여형) | 사용 (서술형) |
|---|---|
| "여러분, 하루에 터미널 몇 번 쓰세요?" | "개발자는 하루에 터미널을 수십 번에서 수백 번 사용합니다." |
| "~해보신 분 계세요?" | "~하는 경우가 많습니다." |
| "따라해보세요." | (제거 — 화면을 보며 자연히 따라함) |
| "여기서 질문 드리겠습니다." | (제거 — 바로 서술로 전환) |
| "~죠?" / "~맞죠?" | "~입니다." |
